This is the experiment page for Laronix Data Recording.

1. Select one example from below, a sound file, with its reference transcription and its speaking rate will be loaded as inputs.
You can check the sound file first and prepare for reading the transcription at a similar tempo.
2. Delete the sound file (click the X button on the right), a recording button will appear.
3. Click the recording button to start, click again to stop. Make sure you are not mispronouncing or including any detectable noises.
4. Click "Submit" button and wait for the result.
5. Please check the message box to see the feedback, if ERROR appears, delete your previous recording and try again :).
6. If "GOOD JOB!" message appears, click "Flag as Perfect" and start another recording.
7. If you try several times (N >= 10) and still can not clear the mission, you can flag your best recording by clicking "Doubtful Speaking Rate" or "Doubtful Naturalness".
Yet this seldom happens, so please try to meet the system's requirement first!
8. If you have any other question, Please contact kevin@laronix.com
